## 部署 Deploy

### 编译 Compile

此项目提供了编写完善的 CMakeLists 文件。您可以采用任意支持它的编译器编译此项目。 但烦请务必注意以下问题：

- 请确认你的环境满足此项目的最低编译标准 C++20 。

- 请**确保编译此项目的设备提供 POSIX 标准的网络头文件库**（例如 Linux / macOS ），否则您可能要自行修改头文件或编译选项。

- 此项目需要第三方依赖库 [SQLite3](https://sqlite.org/index.html) 。
  如果没有在本地设备寻找到可用的 SQLite3 依赖库，编译会回落到使用包含在程序源码中的 SQLite3 3.53.2 源码。
  **此版本程序源码不一定是最新的，可能包含已经被公开的漏洞代码**。

### 运行 Execute

本项目假设服务端和客户端运行在同一机器上，使用本地网址进行访问。
如有需要，请修改 client/Client.h 头文件中的服务器地址，使之与您服务端所部署的真实地址匹配。
